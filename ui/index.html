      The Career Evidence Vault

      Every claim that appears in a tailored resume is traceable to a verified source — a past resume, a LinkedIn export, a GitHub repo, a portfolio page, an uploaded doc. If the engine can’t tie a sentence to evidence, it doesn’t write the sentence. It flags a gap instead.

      Compliance, in plain English

      • We never bypass logins, CAPTCHAs, or rate limits.
      • Scraping sources (LinkedIn / Indeed / Glassdoor) are marked GRAY; we render assisted packets and you submit yourself.

      • You can delete every byte of your data from Settings.
