The Career Evidence Vault

Every claim that appears in a tailored resume is traceable to a verified source: a past resume, a LinkedIn export, a GitHub repo, a portfolio page, an uploaded doc. If the engine cannot tie a sentence to evidence, it does not write the sentence — it flags a gap instead.
